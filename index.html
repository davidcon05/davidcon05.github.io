About me

I'm a Software Development Engineer in Test, specializing in native mobile testing and automation for both Android and iOS. I'm currently working in Raleigh, North Carolina but open to relocating for the right fit.

I enjoy providing automation that supports high velocity development, providing precisison manual testing and accessibility testing. Moreover, as a SDET I try and walk the fine line between advocating for the user and velocity of the team. Relying on native components and accessibility unless it falls short of the mark. My Android Development experience started first with native Android UI test automation, Espresso. This lead to hilt testing for a more robust suite with idling resources and map responses. Next, I found myself fixing bugs and writing PRs for smaller tickets. Now, I build my own Fragment and Compose apps.
